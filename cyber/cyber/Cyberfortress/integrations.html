Integrations

CyberFortress integrates with your existing security stack to provide unified threat detection and response.

Integration Philosophy

CyberFortress is designed to augment, not replace, your existing security tools. We provide native integrations with industry-leading SIEM, EDR, firewall, identity, and cloud platforms.





SIEM Integrations

Splunk

Ingest Splunk alerts, enrich with CyberFortress context, send detection results back to Splunk for centralized logging.


Setup Time: <5 minutes

Elastic

Real-time log ingestion from Elasticsearch. CyberFortress detections indexed back into Elastic for dashboard visualization.


Setup Time: <5 minutes

IBM QRadar

Native QRadar integration via API. Correlate CyberFortress detections with QRadar offenses.


Setup Time: <15 minutes

LogRhythm

Ingest LogRhythm alarms and send CyberFortress threat intelligence back for enhanced correlation.


Setup Time: <15 minutes




EDR Integrations

CrowdStrike Falcon

Receive CrowdStrike detections, trigger isolation actions through CrowdStrike API, correlate endpoint + network + identity data.


Setup Time: <10 minutes

Carbon Black

Query Carbon Black telemetry, execute response actions (isolate endpoint, kill process), enrich threat context.


Setup Time: <10 minutes

SentinelOne

Bi-directional API integration for detection exchange and coordinated response across network and endpoint.


Setup Time: <10 minutes




Firewall Integrations

Palo Alto Networks

Setup Time: <10 minutes

Fortinet FortiGate

Execute firewall policies, update address objects, manage security zones via FortiGate API.


Setup Time: <10 minutes

Cisco Firepower

Integrate with Cisco Firepower for automated policy updates and threat blocking.


Setup Time: <15 minutes

pfSense

Community-contributed integration for pfSense environments via REST API.


Setup Time: <20 minutes

Custom Integrations

REST API

Comprehensive REST API with OpenAPI specification. Integrate CyberFortress with any tool supporting HTTP/REST.


Webhooks

Outbound webhooks for real-time notifications to custom endpoints. Support for authentication and payload customization.


SDKs

Official SDKs available for Python, Go, and JavaScript. Community SDKs for other languages.


Plugin Framework

Python-based plugin framework for custom detection logic, response actions, and data enrichment. Example plugins available on GitHub.
